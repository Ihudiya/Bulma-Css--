In this article we will know what Bulma is all about

Meaning of Bulma Css:

Bulma css is a free popular Open-source frame work which depends on flexbox module. It provides ready-to-use frontend components that you can easily combine to build a responsive web interface.

Bulma is very easy to learn and setup . It adopts a mobile first approach which looks great in small screen size. It is built with SASS (a css pre-processor)

Bulma was created by Jeremy Thomas in the year 2016 and distributed under MIT Lincense, and an MIT lincense is used to linsense the bulma source code. It was released on April 18, 2018. It is now used by more than 200,000 developers.

Features of Bulma css:

  • It is customizable and modualarizable
  • It is a light weight, modern css which is based on framework
  • It contains mobile first styles in a single file, instead of in seperate files.

Advantages of Bulma

  • It provides responsive design for desktop.
  • It is simple to use and learn
  • Bulma is compatible with the majority of recent Browsers
  • It uses minimal Html code which makes it easy to read and write code.
  • Instalation of javascript is not necessary, because it offers a lot of helpful tools.
  • It is pure css framework, so that you can combine with any java script framework such as React Js and Angular Js.

Disadvantages of Bulma

Anything with benefit must also have a drawback. Let's examine that of Bulma

  • Bulma is still in develpment phase
  • It is a new framework wwhich is not as large of a community.
  • It has less documentation and needs some minor improvement.

Differences between Bulma and Other css framework

Both Bulma and Boostrap are free, open source, front-end frameworks, mobile first css framework which do simmilar things, but they both differ in complexity, capability, and popularity.

Bulma Vs Boostrap

If you are a web developer you should be familiar with Boostrap, The most popular css framework. Booostrap has existed for a while before Bulma, So it has a larger user base and support community. This implies that solutions to advanced problems are more likely to be found with Boostrap than with Bulma.However, Bulma's documntation is detailed and straight forward, and its developers community is growing wit time.

Bulma is a css only framework unlike Boostrap. It does not come with Javascript(a scripting language for creating dynamic website Content) or Jquery preinstalled. Javascript can be intimidating for new developers.If u are not yet ready to start learning a new language, Bulma might be a friendlier step into css framework.
